# InternConnect

InternConnect is a mobile application designed to bridge the gap between students seeking internships and companies offering internship opportunities. The app provides a seamless platform for both students and companies to connect, post, and manage internship opportunities.

## üéØ Features

### For Students
- **Account Creation & Profile Management**
  - Create a personal account
  - Set up your profile with educational background and skills
  - Upload your resume/CV
  - Track your application status

- **Internship Search**
  - Browse available internships
  - Filter by location, industry, and duration
  - View detailed internship descriptions
  - Save interesting opportunities
  - View recent internships (posted within the last 7 days)

- **Application Process**
  - Apply directly through the app
  - Track application status
  - Receive notifications about application updates
  - View application history

### For Companies
- **Company Profile Management**
  - Create and manage company profile
  - Add company description and details
  - Upload company logo
  - Manage team access

- **Internship Management**
  - Post new internship opportunities
  - Edit existing internship posts
  - View and manage applications
  - Track applicant status

## üì± How to Use

### For Students

1. **Getting Started**
   - Download and install the app
   - Create a new account or sign in with existing credentials
   - Complete your profile setup

2. **Finding Internships**
   - Use the search bar to find specific opportunities
   - Apply filters to narrow down results
   - Browse through the "Recent" section for new postings
   - Save interesting internships for later

3. **Applying for Internships**
   - Click on an internship to view details
   - Click "Apply" on internships you're interested in
   - Fill out the application form
   - Upload required documents
   - Submit your application

4. **Managing Applications**
   - View all your applications in the "My Applications" section
   - Track the status of each application
   - Receive notifications about updates
   - View application history

### For Companies

1. **Getting Started**
   - Create a company account
   - Complete company profile setup
   - Add company details and logo

2. **Posting Internships**
   - Click "Post New Internship"
   - Fill out internship details:
     - Position title
     - Description
     - Requirements
     - Duration
     - Location
     - Compensation (if any)
   - Review and publish

3. **Managing Applications**
   - View all received applications
   - Filter and sort applications
   - Update application status
   - Communicate with applicants
   - Track applicant progress
